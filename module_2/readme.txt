Основные цели и задачи проекта: Разведывательный анализ данных для модели, которая предсказывала бы результаты госэкзамена по математике.

Краткая информация о данных: Исходный датасет состоит из 30 колонок и 395 строк. Данные содержатся как в числовых, так и номинативных переменных.

Этапы работы над проектом: 
общая оценка датасета;
оценка и очистка данных в числовых столбцах;
работа с неинформативными признаками;
корреляционный анализ;
анализ номинативных переменных;
формирование итогового датасета для построения модели.
